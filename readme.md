# Velvet Coffee

Landing page responsive para una cafetería llamada **Velvet Coffee**.

## Integrantes

- Noelia Barrientos Pacheco
- Yajaira Melissa Alonso Martínez
- Yajaira Rachel Marroquín Alfaro

## Descripción

Velvet Coffee es un sitio web estático tipo landing page diseñado para presentar una cafetería moderna, cálida y elegante. Incluye secciones de inicio, menú destacado, platillos salados, bebidas, postres, actividades, información sobre la marca y contacto.

## Tecnologías utilizadas

- HTML5
- CSS3
- Flexbox
- CSS Grid
- JavaScript
- Git
- GitHub
- GitHub Pages

## Características

- Diseño responsive para celular, tablet y computadora.
- Carruseles de productos y experiencias.
- Sección de contacto con mapa integrado.
- Uso de etiquetas semánticas.
- Diseño visual con paleta de colores cálida.
- Publicación mediante GitHub Pages.
